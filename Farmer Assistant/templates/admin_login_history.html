🔍 Active Sessions Guide

An active session can be identified by a LOGIN event paired with a subsequent LOGOUT event. Use this data to:

  • Monitor peak usage times
  • Detect unusual login patterns
  • Verify user engagement levels
  • Generate session duration reports
